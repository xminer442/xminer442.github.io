Overview

XMiner is High performance and very fast minning.

  • Official Windows support. Small Windows executable, without dependencies.
  • x86/x64 support. Support for backup (failover) mining server.
  • keepalived support. Command line options compatible with cpuminer.
  • CryptoNight-Lite support for AEON. Smart automatic CPU configuration.
  • Nicehash support It's open source software.

You probably understand now why we call UPX the "Ultimate Packer for eXecutables".
